# 🎬 CineAndrade

Um site de streaming de cinema moderno, responsivo e focado na experiência do usuário.

## 🛠️ Tecnologias
* **HTML5:** Estrutura e semântica.
* **CSS3:** Layout (Flexbox/Grid), animações e responsividade.

## 🚀 Funcionalidades
* Design inspirado em plataformas premium.
* Menu de navegação fixo.
* Grade de filmes responsiva.
* Efeitos de hover nos cards de catálogo.
* Layout adaptável para dispositivos móveis.
* Desenvolvido com 🍿 por Lara Silva De andrade
